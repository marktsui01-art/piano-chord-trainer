# User Guide

## Overview
Piano Chord Trainer helps intermediate players practice chords, intervals, and sight-reading using a real MIDI keyboard or microphone.

## Modes

### 1. Lesson Mode
A flashcard-style mode to learn chords.
*   **Features:** View chord notation, play audio, and cycle through standard voicings.
*   **Settings:** Select Key and Mode to explore chords in different contexts (e.g., C Minor, G Dorian).

### 2. Drill Mode
Interactive practice with scoring.

#### Drill Types
*   **Triads & 7th Chords:**
    *   Identify and play the displayed chord.
    *   **Inversions:** Toggle to practice 1st, 2nd, and 3rd inversions.
    *   **Polyphonic Input:** Play all notes at once.
*   **Speed Note:**
    *   Identify the single note displayed.
    *   **Goal:** Reaction time. Validation is octave-independent (C4 accepts C3, C5, etc.).
*   **Interval Recognition:**
    *   Play the start note, then calculate and play the target note.
    *   **Chromatic Mode:** Tests intervals on random roots (e.g., Minor 2nd, Tritone).
*   **Melodic Sight-Reading:**
    *   Play a sequence of notes (e.g., C-D-E-F-G).
    *   **Feedback:** Correct notes flash green; incorrect notes flash red.

## Input Methods

### MIDI Keyboard (Recommended)
Connect a USB MIDI device. The app automatically detects input.
*   **Status:** "MIDI Input connected" will appear in the console/logs.

### Microphone
Use your acoustic piano.
1.  Click **"Enable Mic"**.
2.  Allow browser permissions.
3.  Play clearly. *Note: Currently supports monophonic (single note) detection best.*

### Virtual Piano & Text
*   **Virtual Piano:** Click keys on screen.
*   **Text Input:** Type notes (e.g., "C E G") and press Enter.

## Settings

*   **Key & Mode:** Change the musical context (e.g., Key: A, Mode: Harmonic Minor).
*   **Clef:** Switch between Treble and Bass clef.
*   **Range:**
    *   **Normal:** Standard octave (C4).
    *   **Low:** Shift down one octave.
    *   **High:** Shift up one octave.
    *   **Wide:** Randomizes octave for every question.
